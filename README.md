## 📖 概要

2026年、米国の法廷がSNSの「設計」を裁いた。コンテンツではない。アルゴリズムでもない。**設計思想そのもの**が、子どもの精神的健康を害したとして損害賠償を命じられた。

アテンション・エコノミーは終わりつつある。しかし、次に何が来るのかは、誰にも分かっていない。

本書は、その「次」を描くための本だ。完成されたモデルを提示する本ではない。**設計条件（Design Brief）を提示する本だ。**

アテンション・エコノミーの構造的解剖から始め、法廷が「設計責任」という新しい法的概念を確立した経緯、1兆ドル規模の見えない社会的コスト、アテンション格差という新しい不平等の軸、FacebookのMSI（Meaningful Social Interactions）が証明した「指標の罠」、移行期の実験と失敗、クリエイター報酬の再設計、自己決定理論（SDT）に基づく「真のエンゲージメント」の定義を経て、次世代SNSの6つの設計条件を提示する。

**本書は答えを書かない。設計図を書く。次世代SNSをつくるのは、この設計条件を読んだあなたかもしれない。**
